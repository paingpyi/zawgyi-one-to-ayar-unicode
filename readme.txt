This plugin adds the buttons to burmese unicode fonts ayar to zawgyi vice versa..

== Description ==
Automatic unicode font converter for ayar and zawgyi font for burmese language. <a href="http://www.ayarunicodegroup.com/">Documentation</a>.

The Ayar Unicode Converter feature list:<ul>
<li>Used font embedding methods.</li>
<li>Allow to change ayar font and zawgyi font.</li>
<li>localized interger numbers.</li>
<li>Works on admin dashboard and frondend.</li>
</ul>

== Installation ==
* Just put the plug-in into your plug-in directory and activate it.
